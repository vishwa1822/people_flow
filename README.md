# PeopleFlow — Workforce Intelligence Dashboard

A premium, enterprise-grade Employee Management Dashboard built for HR managers, administrators, and business leaders.

## Features

- **Single-viewport Dashboard** — non-scrollable command center with KPIs, alerts, and team preview
- **Reports** — all workforce charts and analytics in a fixed 2×2 grid
- **Collapsible sidebar** — Dashboard, Employees, Departments, Attendance, Payroll, Reports, Settings
- **Global search** — wired across dashboard and employees views
- **Employee Management** — search, filters, card-based directory
- **Employee Profile** — compact slide-over with personal, work, and performance sections
- **Add Employee** — slide-over with dynamic department-specific fields

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Build

```bash
npm run build
npm run preview
```
